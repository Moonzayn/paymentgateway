# PPOB Express 💰

Aplikasi bisnis PPOB (Payment Point Online Bank) terlengkap dengan fitur all-in-one untuk menjalankan usaha pulsa, token listrik, voucher game, dan layanan digital lainnya.

---

## ✨ Keunggulan

| Keunggulan | Deskripsi |
|------------|-----------|
| **All-in-One** | Satu aplikasi untuk semua kebutuhan PPOB |
| **Mudah Digunakan** | Interface intuitif, mudah dipelajari |
| **Multi-Produk** | Pulsa, Kuota, Token Listrik, Game, E-Money |
| **Multi-User** | Sistem keagenan dengan role yang fleksibel |
| **POS Terintegrasi** | Kasir modern dengan QRIS support |
| **Mobile Ready** | REST API untuk konek ke aplikasi mobile |
| **Keamanan Terjamin** | 2FA, password policy, audit log |

## 📱 Mobile API

REST API yang ready untuk integrasi dengan:
- Aplikasi Android (Flutter)
- Aplikasi iOS
- PWA (Progressive Web App)

```
Base: /api/v1/

• POST /auth/login      → Login
• GET /user/saldo       → Cek saldo
• GET /pos/products     → Produk POS
• POST /pos/transactions → Transaksi POS
• GET /chat/messages    → Chat support
```
